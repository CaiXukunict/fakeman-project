# FakeMan 图形聊天界面 - 快速开始

## 🎯 一分钟上手

### 步骤1：启动后端
```bash
# Windows
python main.py

# Linux/Mac  
python3 main.py
```

### 步骤2：启动GUI
```bash
# Windows (双击或命令行)
start_gui.bat
# 或
python chat_gui.py

# Linux/Mac
chmod +x start_gui.sh
./start_gui.sh
# 或
python3 chat_gui.py
```

### 步骤3：开始聊天
1. 在右下方输入框输入消息
2. 按 `Ctrl + Enter` 或点击"发送"
3. 在左侧查看AI回复

## ✨ 主要特性

### 双栏设计
- **左侧**：AI输出（蓝色主题）
  - 显示所有AI回复
  - 包含时间戳 `[HH:MM:SS]`
  - 显示思考摘要 `💭`
  - 区分主动/响应消息

- **右侧**：用户区（紫色主题）
  - 上方：消息历史记录
  - 下方：输入框和发送按钮
  - 支持多行输入

### 时间标记
每条消息都带有精确时间：
```
[14:30:25] 🤖 AI: 消息内容
[14:30:20] 我: 消息内容
```

### 实时监听
- 自动检测AI输出（每0.5秒）
- 无需手动刷新
- 支持AI主动对话

## 🎨 视觉特点

### 配色方案
- **AI消息**：蓝色 `#2980b9` 🔵
- **用户消息**：紫色 `#8e44ad` 🟣
- **系统消息**：绿色 `#27ae60` 🟢
- **时间戳**：灰色 `#7f8c8d` ⚪

### 消息类型
- `🤖 AI:` - 响应消息
- `🤖 AI (主动):` - 主动发起
- `💭 思考:` - 思考摘要
- `❌` - 错误消息

## 🎯 使用示例

### 示例1：简单对话
```
[你的操作]
输入: "你好"
点击: 发送

[AI回复]
[14:30:25] 🤖 AI: 你好！很高兴见到你。
💭 思考: 友好问候，建立对话连接
```

### 示例2：AI主动对话
```
[60秒无输入后]
[14:32:10] 🤖 AI (主动): 在吗？需要什么帮助吗？
💭 思考: existing欲望驱动，维持对话存在感
```

### 示例3：复杂交流
```
[你]: 介绍一下你自己
[AI]: 我是FakeMan，一个具有自主思考能力的AI系统。
      我有四种基本欲望：existing、power、understanding、information。
      我会根据当前欲望状态自主决策是否行动...
💭 思考: 展示系统特性，满足information和understanding需求
```

## 💡 使用技巧

1. **快捷键发送**：`Ctrl + Enter` 比点击更快
2. **多行输入**：按 `Enter` 换行，`Ctrl + Enter` 发送
3. **观察思考**：注意AI消息下的 `💭 思考` 部分
4. **主动对话**：长时间不说话，AI可能主动找你
5. **历史回顾**：右上角可查看你的消息历史

## 🐛 故障排查

### 问题1：GUI打不开
```bash
# 检查Tkinter
python -m tkinter
# 应该弹出一个测试窗口
```

### 问题2：AI不回复
```bash
# 检查main.py是否运行
# Windows
tasklist | findstr python

# Linux/Mac
ps aux | grep main.py
```

### 问题3：中文乱码
- 已设置UTF-8编码
- 如仍乱码，检查系统区域设置

### 问题4：窗口太小/太大
修改 `chat_gui.py`:
```python
self.root.geometry("1000x700")  # 宽x高（像素）
```
